C++ 操作符的优先级 特别注意：同一个优先级的操作符里的优先级是根据结合性来排序的。 一般结合性都是从左到右。除了三目运算符‘？：’和赋值运算符‘=’，还有就是优先级为2的各种操作符 例如*++p是从右到左的。 优先级 |操作符 |描述 |例子 |结合性 1 |() |调节优先级的括号操作符 |(a + b) / 4; |从左到右 |[] |数组下标访问操作符 |array[4] = 2; | |-> |通过指向对象的指针访问成员的操作符 |ptr->age = 34;
